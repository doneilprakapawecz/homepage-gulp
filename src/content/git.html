git branche

>git branch //pokazuje na której gałęzi gita jesteśmy
>git branch "nazwa"//tworzy nową gałąź gita
>git checkout nazwa//przechodzimy na gałąź "nazwa"
>gitk //otwieramy narzędzie do obserwowania commitów
>git push --set-upstream origin "nazwa"//tworzymy nową gałąź na githubie
>git checkout master// wracamy na główną gałąź
>git merge nazwa// zmiany z gałęzi "nazwa"zapisujemy do głównej gałęzi "master" !polecenie wykonujemy z gałęzi "master"
